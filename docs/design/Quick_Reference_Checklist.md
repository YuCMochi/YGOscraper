## 💡 有用的連結和命令

### 常用命令
```bash
# 建立新分支
git checkout -b feat/frontend-redesign

# 安裝依賴
npm install zustand @tanstack/react-query axios
npm install -D vitest @testing-library/react

# 開發服務器
npm run dev

# 執行測試
npm run test

# 檢查無障礙
npm run axe  # (需要配置)

# 檢查效能
npm run lighthouse  # (需要配置)
```
